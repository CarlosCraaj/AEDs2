LEGENDA: Cara sublinhado ( _ ) representa 1 espaço.

Enunciado: Represente graficamente o código C abaixo
Código: 
Elemento** e4 = (Elemento**) malloc(3*sizeof(Elemento*));
e4[0] = (Elemento*) malloc(sizeof(Elemento*));
e4[2] = (Elemento*) malloc(sizeof(Elemento*));

Resposta:
e4 -> _ _ _
      | | |
      _ ? _

para outro elemento não especificado.
