LEGENDA: Cara sublinhado ( _ ) representa 1 espaço.

Enunciado: Represente graficamente o código C abaixo
Código: 
Elemento** e4 = new Elemento*[3];
e4[0] = new Elemento;
e4[2] = new Elemento;

Resposta:
e4 -> _ _ _
      | | |
      _ ? _
Os elementos 0 e 2 apontam para outros endereços, e o elemento 1
para outro elemento não especificado.
